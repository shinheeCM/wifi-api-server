# wifi-api-server

```
    git add .
    git commit -m "server.js"
    git remote add origin https://github.com/shinheeCM/wifi-api-server.git
    git push -u origin main
    git checkout main
    git pull
```

# run node
```
    npm i
    sudo npm start
```
